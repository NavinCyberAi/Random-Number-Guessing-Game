# 🎯 Random Number Guessing Game

A simple Python project where the player guesses a randomly generated number between **1 and 100**. Each game creates a new random number, making every play different.

## ✨ Features

- Random number generated every game
- Number range: 1 to 100
- Maximum of 10 attempts
- Hints after each guess
- Handles invalid input
- Displays the correct answer and total attempts

## 🛠️ Technologies Used

- Python 3
- Random Module

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/NavinCyberAi/Random-Number-Guessing-Game.git
```

2. Navigate to the project folder.

```bash
cd Random-Number-Guessing-Game
```

3. Run the program.

**For Linux/macOS:**

```bash
python3 Number_Guessing.py
```

**For Windows:**

```bash
python Number_Guessing.py
```

> **Note:** On Windows, use `python` if `python3` is not recognized. On Linux and macOS, `python3` is commonly used.
## 🎮 How to Play

1. Run the program.
2. Enter your name.
3. Guess a number between **1 and 100**.
4. Follow the hints to find the correct number.
5. Guess the number within **10 attempts**.

 ## 🎯 Purpose

This Python game project was created to practice Python programming, improve logical thinking, and develop skills in Object-Oriented Programming (OOP), loops, conditional statements, exception handling, random number generation, and problem-solving.
